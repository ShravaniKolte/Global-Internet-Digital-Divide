# Global-Internet-Digital-Divide

# Global Internet Access & Digital Divide Tracker

## Overview
Analysis of global internet penetration across 237 countries (2010–2024),
uncovering digital inequality patterns and predicting connectivity by 2030.

## Tools Used
Python | Pandas | Matplotlib | Seaborn | Scikit-learn | Power BI | SQL

## Dataset Sources
- World Internet Usage Data (Kaggle)
- Global Internet Usage (Kaggle)
- Internet Users Per 100 People (World Bank)
- Global Economic Indicators 2015–2024 (Kaggle)
- Wikipedia World Statistics 2023 (Kaggle)

## Project Phases
- [x] Week 1 — Data cleaning & merging
- [x] Week 2 — Exploratory Data Analysis
- [ ] Week 3 — ML Model (Classification + Regression)
- [ ] Week 4 — Power BI Dashboard + Final Report

## Key Findings (EDA)
- Europe leads internet penetration at 91.5%
- Africa trails at 42.6% — biggest digital divide
- HDI has 0.92 correlation with internet access
- Global penetration grew from 34.5% (2010) to 48.6% (2015)

## Digital Divide Classification
| Status | Definition |
|---|---|
| Connected | ≥ 75% penetration |
| Catching Up | 45–74% penetration |
| Left Behind | < 45% penetration |
